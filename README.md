# Quantium Retail Analytics

## Project Overview

This project analyzes retail transaction data to understand customer purchasing behaviour, evaluate the performance of a trial store, and generate data-driven business recommendations. The analysis demonstrates how customer insights and sales data can support strategic retail decision-making.

---

## Business Problem

A retail client wants to better understand its customers and determine whether a selected trial store performed significantly better than comparable control stores after implementing a new strategy.

---

## Project Objectives

- Analyze customer purchasing behaviour.
- Identify high-value customer segments.
- Evaluate trial store performance.
- Compare trial and control stores.
- Present actionable business recommendations.
